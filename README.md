# Pure Java 3D Renderer

A 3D software renderer written entirely in Java and built from scratch to explore the fundamentals of computer graphics.

This project focuses on core rendering concepts such as 3D transformations, projection, and the graphics pipeline, without using external rendering engines. It was developed as a learning project to better understand how 3D scenes are converted into 2D visual output through custom Java code.

## Features

- Built entirely in pure Java
- Custom 3D transformation pipeline
- 3D to 2D projection
- From-scratch rendering logic
- Educational graphics project

## Future Plans

- Add lighting and shading
- Improve camera controls
- Add depth buffering
- Support more complex models
